Responsive Web Design Lab

Lab Title

Responsive Webpage Design & Layout Experiment

Description

This lab demonstrates the creation of a fully responsive webpage using HTML, CSS, and media queries. The goal is to test how layout, typography, and components adapt across different screen sizes.

Steps to View Responsive Behaviour

1.	Open the project in any browser.
2.	Right-click → Inspect → Toggle Device Toolbar.
3.	Test the webpage at the following breakpoints:
   
      o	320px – 480px (Mobile)
   
      o	481px – 768px (Small Tablets)
   
      o	769px – 1024px (Large Tablets / Small Laptops)
   
      o	1025px and above (Desktop)

Technologies Used

•	HTML
•	CSS
•	Media Queries
